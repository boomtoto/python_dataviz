![alt text](miniconda2.png)

 3. ให้เลือกระบบปฏิบัติการที่ตรงกับเครื่องที่ท่านใช้งานอยู่ แล้วทำการดาวน์โหลดจะได้ดังภาพข้างล่าง

![alt text](miniconda5.1.png)

![alt text](miniconda6.png)
